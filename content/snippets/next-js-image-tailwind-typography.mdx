---
title: "Next.js Images with Tailwind Typography"
description: "Fix the aspect ratio of images in MDX"
---

When using the [Next.js Image component](https://nextjs.org/docs/basic-features/image-optimization) in MDX with [Tailwind Typography](https://github.com/tailwindlabs/tailwindcss-typography) your image might appear with an incorrect aspect ratio like this.

```jsx
<Image
  src="/snippets/next-js-image-tailwind-typography/example.jpg"
  width={1200}
  height={630}
/>
```

<Image
  src="/snippets/next-js-image-tailwind-typography/example.jpg"
  width={1200}
  height={630}
  alt="Stretched example"
  title="Stretched example"
/>

To fix it all you need to do is to add the [object-contain](https://tailwindcss.com/docs/object-fit#contain) class.

```jsx {5}
<Image
  src="/snippets/next-js-image-tailwind-typography/example.jpg"
  width={1200}
  height={630}
  className="object-contain"
/>
```

  src="/snippets/next-js-image-tailwind-typography/example.jpg"
  width={1200}
  height={630}
  className="object-contain"
  alt="Fixed example"
  title="Fixed example"
/>

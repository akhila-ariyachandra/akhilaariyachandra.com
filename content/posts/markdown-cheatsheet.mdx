## Lists

---

### Ordered List

```markdown
1. First Item
2. Second Item
3. Third Item

OR

1. First Item
1. Second Item
1. Third Item
```

1. First Item
2. Second Item
3. Third Item

### Unordered List

```markdown
- First Item
- Second Item
- Third Item

OR

- First Item
- Second Item
- Third Item

OR

- First Item
- Second Item
- Third Item
```

- First Item
- Second Item
- Third Item
